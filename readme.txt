Implementations of a basic ToDo app

Design:
- textbox to enter todo text
- button next to textbox to create todo, alerts instead if text field is blank
- list of todos
- each todo has a checkbox to mark/unmark, button to delete
- save/load buttons to save and load list of todos to/from server
- on page load, list of todos is retrieved from server and displayed

Implementations:
- Pure javascript
- jQuery
- Angular2
- Backbone
- React
